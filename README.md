# 📚 AI Flash Cards – Focus on Learning

![AI Studio](https://media2.dev.to/dynamic/image/width=1000,height=420,fit=cover,gravity=auto,format=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2F1waok8imyjp2d9ozcnim.png)

AI Flash Cards is an AI-powered learning application that converts textbook images into structured flashcards and quizzes using Google Gemini.

---

## 🚀 Features

* 📸 Scan textbook images
* 🤖 Generate flashcards using AI
* 🧠 Create quizzes with explanations
* ⏱️ Focus mode for better study sessions
* 📂 Organize study materials
* 🔗 Share study content

---

## 🛠️ Tech Stack

* React + TypeScript
* Tailwind CSS
* Google Gemini API
* Firebase (for storage & database)

---

## ▶️ Run Locally

**Prerequisites:** Node.js

1. Install dependencies:

```
npm install
```

2. Create a `.env.local` file and add your Gemini API key:

```
VITE_GEMINI_API_KEY=your_api_key_here
```

3. Start the development server:

```
npm run dev
```

4. Open the local URL shown in the terminal (e.g., `http://localhost:xxxx`)
